## Основные

### Обобщённый блок

`block`

### Страница

`page`

### Тело блока (например, текст с изображением)

`content`, `body`, `main`

### Верхняя часть блока (например, заголовок)

`header`, `head`

### Нижняя часть блока (например, дата добавления, категории и т.д.)

`footer`

### Основной раздел

`primary`

### Вторичный раздел

`secondary`

### Боковая колонка страницы, содержащая вспомогательный контент (виджеты, рекламные блоки и т.п.)

`sidebar`, `aside`

### Один из разделов контента

`section`

### Один из подразделов контента

`subsection`

### Блок навигации

`navigation`, `nav`

### Меню

`menu`

### Ссылка

`link`

## Списки

### Список

`list`, `items`

### Пункт списка, один из повторяющихся элементов

`item`, `element`, `part`

### Номер повторяющегося элемента (часто используется в качестве БЭМ-модификатора)

`number`

`outer`

### Внутренняя обёртка

`inner`

### Контейнер

`container`, `box`, `holder`

`row`
